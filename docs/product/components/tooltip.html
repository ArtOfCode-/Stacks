{% header {element: 'h2', text: 'Accessibility'} %}
Item Applied Description
aria-controls="[id]" Trigger Identifies the tooltip the item controls. (Source)
aria-hidden="[state]" Trigger, Tooltip Indicates to screen readers if a tooltip should be visible. (Source)
aria-describedby="[id]" Trigger Indicates which tooltips provides further information about the item. (Source)
aria-haspopup="[state]" Trigger Indicates if the item has a tooltip popup attached to it. (Source)
aria-expanded="[state]" Trigger Indicated if the item's tooltip is currently expanded or collapsed. (Source)
role="tooltip" Tooltip To help with screen readers, applied to each tab item. (Source)
data-method="[method]" Tooltip Custom data value for stating how the tooltip should be shown: hover or click.
id="[id]" Tooltip This should match the same value in the trigger's aria-describedby and aria-controls values.
