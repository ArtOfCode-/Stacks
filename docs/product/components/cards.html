{% header {element: 'h2', text: 'Box Shadows'} %}

By default cards are indifferent about box shadows. No box shadows are added by default. If you want your card to have a shadow, apply one of the .bs-* classes. When applying a .bs-* class with a .s-card component, a :hover style is added to the base .bs-* box shadow.
